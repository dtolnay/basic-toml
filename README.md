A library for parsing and producing data in [TOML] format using [Serde].

This crate is a stripped down fork of version 0.5 of the `toml` crate (from
before the `toml_edit` rewrite).

[TOML]: https://toml.io
[Serde]: https://serde.rs

```toml
[dependencies]
basic-toml = "0.1"
```
